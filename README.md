Server
- Open backend folder in visual studio
- Voer command uit in Package Manager Console: Update-Database
- Start IIS Express(f5)

Client
- Open frontend folder in visual studio code
- Open een terminal en zorg dat je in de juiste map staat
- Voer command uit: npm install
- Voer command uit om client te starten: ng serve
- Voer command uit om testen te starten: ng test
- Voer command uit om protractor te starten: ng e2e
